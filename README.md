### Installation
Install nvm.
https://github.com/creationix/nvm

```sh
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash
```


Install StrongLoop / LoopBack
```sh
$ npm install -g strongloop
```

```sh
$ npm start
```
